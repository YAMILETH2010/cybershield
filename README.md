# Escudo Cibernético 🛡️

**Escudo Cibernético** es una plataforma web educativa e informativa dedicada a la ciberseguridad. Su misión principal es concienciar a usuarios comunes, profesionales y pequeñas empresas sobre los riesgos digitales actuales, proporcionando herramientas prácticos, guías de mitigación y mejores prácticas para proteger su identidad y activos en el entorno digital.


## 🚀 Características Principales

- **Centro de Aprendizaje:** Artículos interactivos, tutoriales y guías paso a paso sobre seguridad digital (gestión de contraseñas, doble factor de autenticación, detección de phishing, etc.).
- **Simulador de Phishing:** Una sección interactiva para que los usuarios aprendan a identificar correos y enlaces fraudulentos en un entorno seguro.
- **Auditor de Contraseñas:** Herramienta local (client-side) para verificar la robustez de las contraseñas sin comprometer la privacidad.
- **Alertas de Vulnerabilidades en Tiempo Real:** Feed actualizado con las últimas amenazas, brechas de datos masivas y parches de seguridad críticos.
- **Sección de Recursos:** Plantillas de políticas de seguridad para pymes y guías descargables en PDF.


## 🛠️ Tecnologías Utilizadas

La plataforma está construida utilizando tecnologías modernas, eficientes y seguras:

- **Frontend:** HTML5, CSS3 (con metodologías BEM), JavaScript Moderno (ES6+).
- **Estructura y Estilos:** Preprocesador Sass / Tailwind CSS para un diseño responsivo y mobile-first.
- **Interactividad y Estado:** React.js (o Vanilla JS según la arquitectura final) para los módulos del simulador y auditor.
- **Seguridad del Sitio:** Implementación estricta de políticas de seguridad (CSP, HTTPS, protección contra XSS y CSRF).
